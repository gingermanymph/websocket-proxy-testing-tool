WebSocket Proxy & Testing Tool

Your ultimate Chrome Extension for WebSocket monitoring, testing, and data manipulation.

Get Started Download Chrome Extension

Features

WebSocket Proxy

  • Monitor, log, and view data between web client and server
  • Modify data before sending using JavaScript code editor
  • Modify data before receiving using JavaScript code editor (Coming Soon)

WebSocket Client

  • Connect to WebSocket servers (Send, Receive, Monitor, Log, View data)
  • Send text/JSON using a text/code editor
  • Generate objects dynamically using JavaScript code editor

WebSocket Security Check

Feature coming soon!

Roadmap

  • Introduce WebSocket Security Checks (Q2 2025)
  • Expand WebSocket Client capabilities with additional tools (Q3 2025)
